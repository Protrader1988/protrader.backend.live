üíª Example API Calls

Get Portfolio (JavaScript)

fetch('/api/portfolio/')
  .then(res => res.json())
  .then(data => console.log(data));

Get Historical Data (cURL)

curl "https://protrader-backend-web.onrender.com/api/history/?symbol=AAPL&timeframe=1Min&limit=100"

Place Order (Python)

import requests

response = requests.post('/api/order/', json={
    "symbol": "AAPL",
    "qty": 10,
    "side": "buy",
    "type": "market"
})

Run Backtest (JavaScript)

fetch('/api/backtest/', {
  method: 'POST',
  headers: {'Content-Type': 'application/json'},
  body: JSON.stringify({
    symbol: 'AAPL',
    timeframe: '1Min',
    limit: 500
  })
}).then(res => res.json());
